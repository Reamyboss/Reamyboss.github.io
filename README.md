# Reamyboss.github.io
🛡️ Shield Matrix Engine

AI-powered sports prediction SaaS — built from scratch, zero paid APIs.

What Is This?
Shield Matrix Engine is a full-stack sports prediction platform that scrapes data from
public sources, runs a Poisson probability model, and serves predictions through a
REST API consumed by a Flutter mobile frontend.
No paid APIs. No third-party keys. Just Python, math, and public data.

How It Works
Public Sources (Wikipedia, Wikidata, RSS)

Prediction Model
Shield Matrix uses a Poisson distribution model to predict match outcomes:

Calculates each team's attack strength and defence weakness
Derives expected goals for home and away teams
Generates win/draw/loss probabilities using Poisson math
Returns percentage likelihood for each outcome

Setup & Installation
Prerequisites

Python 3.10+
Flutter 3.0+
Git

Backend
bash# Clone the repo
git clone https://github.com/Reamyboss/shield-matrix-engine.git
cd shield-matrix-engine

# Create virtual environment
python -m venv venv
venv\Scripts\activate        # Windows
source venv/bin/activate     # Mac/Linux

# Install dependencies
pip install -r requirements.txt

# Run the server
uvicorn main:app --reload --port 8000
Visit http://localhost:8000 — API is live.
Visit http://localhost:8000/docs — interactive API docs (auto-generated by FastAPI).
Flutter Frontend
bashcd shield_matrix
flutter pub get
flutter run
Run Data Collector Manually
bashpython collector.py
